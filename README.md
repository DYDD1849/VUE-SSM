# web-SSM

#### 介绍
学生成绩管理系统

#### 软件架构
前后端分离
vue+ssm框架


#### 安装教程
1. 安装npm包管理器
2. 安装@vue/cli 5.0.8
3. 安装mysql 8.0
4. 安装9.0及以上tomcat
5. 安装idea
#### 使用说明

1. 前端运行前执行npm install命令
2. 后端运行前导入数据库文件，确保mysql相关配置一致
3. vue运行：npm run serve
4. tomcat默认项目路径（不能写错，这个和前端是一致的）http://localhost:8080/back_ssm/
5. idea配置后端url路径上下文一定是“http://localhost:8080/back_ssm/”
6. 后端启动tomcat即可
